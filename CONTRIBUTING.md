# Contributing to EdgePicks

Thank you for your interest in improving EdgePicks! Contributions of all sizes are welcome.

## Getting Started
- Fork the repository and clone your fork.
- Install dependencies with `npm install`.

## Development Workflow
- Create a new branch for each change.
- Make your modifications and include documentation when appropriate.
- Run the test suite with `npm test` and ensure it passes.

## Commit Guidelines
- Use concise commit messages that describe your changes.
- Keep each pull request focused on a single topic.

## Reporting Issues
- Use GitHub issues to report bugs or request features.
- Provide clear steps to reproduce and relevant context.
